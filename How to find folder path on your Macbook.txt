
Please refer to the procedures to copy file path and find your working folder/working directory of Stata.

Method 1:
Right-click the your file to open a menu
Then, hold alt ⌥ key down
Click Copy "Your item" as Pathname
Then, you can paste the item path as text.

Method 2:
View (Top of Screen)->Show path bar->See the path underneath your folder browser-> right/control click to copy folder path

Method 3:
You can also use the option+⌘+C key combination to copy the full path to the clip board. You can then paste this full path as text.
